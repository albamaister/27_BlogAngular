# BlogAngular

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.9.

FrontEnd of ApiRestLaravel: SPA web application type blog

- Models
- Components
- Routing
- User Registration
- Login in the front end
- User settings
- Ticket management
- List of entries
- Image upload
- Guards and permits
